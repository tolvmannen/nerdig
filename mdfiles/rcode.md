## RCODE - Response Code
---
<br/>

This 4 bit field is set as part of responses.


RCODE    | Name   
:-------|:------------
 0      | NOERROR 
 1      | FORMERR
 2      | SERVFAIL
 3      | NXDOMAIN
 4      | NOTIMP
 5      | REFUSED


For at full list of possible RCODEs, se documentation at 
[IANA](https://www.iana.org/assignments/dns-parameters/dns-parameters.xhtml#dns-parameters-6)
